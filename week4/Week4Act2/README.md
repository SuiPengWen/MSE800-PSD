# Week 4 - Activity 2: OOP Project for Area and Perimeter

## Project Overview
This is an object-oriented programming (OOP) project developed for Week 4 Activity 2. The program allows users to input the length and width of a rectangular piece of land, then calculates and displays its area and perimeter using Python.

## Project Structure
The project consists of two `.py` files:
1.  `rectangle.py` - Contains the `Rectangle` class with methods for area and perimeter calculation.
2.  `main.py` - The main program that handles user input, creates an instance of `Rectangle`, and displays the results.

## Features
- Uses OOP principles with a dedicated `Rectangle` class.
- Implements two core methods:
  - `calculate_area()`: Computes the area of the rectangle.
  - `calculate_perimeter()`: Computes the perimeter of the rectangle.
- Accepts user input for dimensions.
- Displays clear output for calculated results.

## How to Run
1.  Ensure you have Python 3 installed on your machine.
2.  Download both files into the same folder.
3.  Open a terminal/command prompt and navigate to the project folder.
4.  Run the program with the command:
    ```bash
    python main.py
