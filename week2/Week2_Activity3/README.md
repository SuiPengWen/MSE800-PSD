# Week 2 - Activity 3: OOP Calculator Project
## Project Description
This project is an upgraded version of Week 2 Activity 1 using **Object-Oriented Programming (OOP)**.

## OOP Concepts Used
- 1 Class: `Calculator`
- 3 Methods: `basic_operation()`, `factorial()`, `show_result()`
- 2 External Functions: `welcome_message()`, `run_calculations()`
- Supports complex numbers, integers, float
- Supports +, -, *, /, %, and factorial

## How to Run
Run the file: oop_calculator.py
